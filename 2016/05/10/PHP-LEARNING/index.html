分支结构

单一条件分支结构——>if单一的条件语句

例：

if（$date<20）{

echo "have a good day!";

}

双向分支结构——>if else语句

例：

if（$date<20）{

echo "have a good day!";

}else {

echo "have a good night!";

}

多向条件分支结构——>if elseif else

例：

if（$date<10）{

echo "have a good morning!";

}elseif（$date<20） {

echo "have a good day!";

}else{

echo "have a good night!";

}
