PHP array data structure

0x00前言

数组是数据类型之一，是存放数据的容器，可以实现其他编程语言中的堆栈列队等数据结构的功能；批量处理数据！

0x01正文

数组的定义：

直接赋值声明；

使用$arr();声明二

数组的分类：

索引数组（枚举数组），关联数组，重载数组；

数组的遍历：

1、for语句遍历数组；

2、foreach语句遍历数组；

与定义数组：

宝库来自于web服务，用户输入的数据；

不用声明，在每个php脚本中已经默认存在，定义变量的时候尽量避免和预定的全局变量同名

在全局范围内自动生效，在函数中直接就可以使用，

$_SERVER web服务器设定或者直接与当前脚本环境相关联

$_EVN 执行环境直接提交至脚本的变量

$_GET 直接由URL请求提交至脚本的变量

$_POST 由HTTP POST方法提交至脚本的变量

$_REQUEST 由 GET POST COOKIE方式提交至脚本，不常用，信用等级低

$_FILES 由HTTP POST文件上传提交至脚本的变量

$_COOKIE 由HTTPcookies方法提交至脚本的变量

$_SESSION 当前注册脚本会话的变量

$GLOBALS 包含上述所有的有效变量，该数组的键名为全局变量的名称

合并数组：

用array_merge合并数组成为联合数组；可以重载数组

连接数组：

拆分数组：

从offset偏移位置开始，到offset+length位置结束（下标位置）

array array_slice（）

接合数组：

array_splice($a,4)

数组的交集：

array_intersect()

关联数组的交集：

键和值都相同才输出值

数组的差集：

array_diff

关联数组中的差集：

获取当前数组的键：

使用while结合key($arr )遍历数组

current（）获取当前指针指向的值

array each（）配合list获取键值对

查找、筛选、

in_array():搜索特定值找到返回真，找不到返回假；

array_key_exits()判断键是否存在，存在返回真，找不到返回假

array_search():判断存不存在，还能找到下标

array_keys():将返回一个数组，包括返回键和键值

array_values():返回一个数组中的所有值，并自动为返回的数组提供索引的值

列队有两种形式：先进先出（FIFO）,后进先出（LIFO）

在数组头部添加元素

array_unshift( )

在数组末尾添加元素

array_push( )

从数组头部删除元素

array_shift( )

从数组尾删除元素

array_pop()​

附加​

数组就是一组有序数据的集合，变量是单个数值的容器，数组就是多个数值得容器；

JS中的数组创建：

var arr=new Array(10,20,30);//JS里边必须有这一步

arr[4]=40;

arr[5]=50;//追加数组元素

PHP中数组的创建：

$arr=array(10,20,30);

$arr[]=20;//如果没有指定下标就在最大下标下边+1

数组的分类：枚举数组，关联数组，多维数组

（1）枚举数组：和JS一样

下标从零开始的正整数

$arr[1]=10;

$arr[2]=20;

………

（2）关联数组：

下标是字符串

$arr[“name”]=”张三”；

$arr[“sex”]=”男”；

$arr[“age”]=”15”;

……………..

数组的创建：

1、使用[ ]创建

$arr[0]=10;如果数组不存在，就创建一个数组！

JS中的数组创建：

var arr=new Array[10,20,30];//JS里边必须有这一步

arr[4]=40;

arr[5]=50;//追加数组元素

PHP中数组的创建：

$arr=array[10,20,30];

$arr[]=20;//如果没有指定下标就在最大下标下边+1

2、使用array创建数组

JS中的数组创建：

var arr=new Array(1,2,3,4,5,6,7);

arr[4]=50;

………

PHP中创建数组：

$arr=array(1,2,3,4,5,6,7);

使用print_r（$arr）遍历数组

重载数组下标：重载符号就是‘=>’

重新设置元素的下标使用“=>”运算符；

$arr = array(

“name”=>10,

10=>100,

);

在数组中键名就是指下标（数字或字符），键值就是数据；
