0x01.2016.04.10
上午学习了环境的搭建，整个环境分为五部分，用户、企业、公网、h3c交换机、实验客户机、整个网络环境在企业和实验客户机与交换机连接的出口处设置防火墙，这个地方应给上个图会更清晰，先大概了解一下吧；

0x02.2016.04.11
在虚拟机中安装centos，进行了基础的配置，网络配置、mysql的安装、apache安装、php安装
接下来直接网络配置： 桥接模式下
配置ip：
ifconfig （查看适配器）
ifconfig -a （查看所有的适配器）
ifconfig eth0 up （其中eth0是网卡名字 ；激活网卡）
ifconfig eth0 ip netmask 子网掩码（配置ip以及子网掩码）
route add default gw 网关（配置网关）
注意：在配置ip的时候先查看主机的ip配置在同一网段中，ip不能相同,否则会出现冲突，在使用远程连接虚拟机的软件的时候无法连接！！！
配置DNS：
cd /etc/ （切换到etc目录下）
ls （查看当前目录下的文件）
vi resolv.conf（找到这个文件对其进行编辑）
例如： nameserver 114.114.114.114
nameserver 8.8.8.8
配置一个百度和谷歌的dns
init 6或者/etc/init.d/network restart 重启网卡！
接下来ping一下网关以及百度,基本上就配置完成了！

NAT连接方式下：
手动配置网络就不说了，可以仿照着桥接模式配置一下
需要注意的是：需要在虚拟机上边的“编辑”点击进入“虚拟机网络配置”中点NAT配置查看一下网关！
接下来就说一下如何文件配置网络：
cd
/etc/sysconfig/network-scripts/
（进入到这个目录下）
vi ifcfg-eth0
（修改这个配置文件）
将ONBOOT=NO修改为ONBOOT=YES
（自动启动配置）
将BOOTPROTO=dhcp修改为BOOTPROTO=static
（将动态分配ip改为静态）
接下来配置指定的ip，子网掩码，网关:
IPADDR=
NETMASK=
GATEWAY=
一切配置完毕之后，重启网卡:
service network restart
注意：配置dns最好是在etc/resolv.conf文件下配置，另外桥接手动配置以后不要重启网络；

0x03.2016.04.12
磁盘的挂载分区：
前期准备需要在centos关闭的时候重新添加一个硬盘！！！
直接上命令:
fdisk -l 查看磁盘信息
df -lH 查看磁盘挂载信息
fdisk /dev/sdb 创建分区
m 帮助查看参数信息
n 添加一个新分区
1 创建一个分区
p 主分区
w 写入分区
mkfs -t ext3
/dev/sdb1 格式化分区
mkdir
/home/jia 创建挂载目录
mount
/dev/sdb1 /home/jia
df -lH 查看挂载信息
注意：这种挂载系统重启以后，挂载信息会丢失，若要不丢失需要将挂载信息添加到/etc/fstab文件中！！！

0x07.2016.04.16
ifconfig的使用：
ifconfig 查看网卡信息
路由规则的添加：
route -C -n -C缓存
route add 0.0.0.0(default) gw 192.168.111.2添加默认网关
route del default gw 192.168.111.2删除默认网关
route add -net 192.168.111.132 netmask 255.255.255.0 dev eth0 添加网络地址dev指定出去的网卡
_strip去除命令尾部的回车之类的命令

nslookup 查看dns
解析记录：
A记录 IP地址记录，把一个域名解析到某一个对应的IP
CNAME记录 别名指向，用子域名代替IP地址（如果修改IP地址，只需要改变子域名的解析即可）
NS记录 解析服务器记录 用来表明由哪台服务器对该域名进行解析，这里的NS记录只对子域名生效。
AAAA记录 将域名解析到ipv6地址的DNS记录，也可以将子域名解析到ipv6地址上；
TXT记录 为某个主机或域名设置联系信息

Linux下dns解析：dig host nslookup
由于迷你版的centos上边没有安装nslookup需要进行安装：
安装nslookup
yum search bind-utils
yum install bind-utils
yum install
按照步骤安装就可以了！

dig 查询地址
dig 域名 查询A记录
dig 域名 ns 查询NS记录
dig 域名 +trace dig追踪域名的解析过程

host 追踪地址
host 主机名 显示主机名的主机地址
host IP 显示地址为IP的主机名
host -n -t mx 域名 显示域名的域的MX记录

nslookup 查询记录
nslookup IP 查询PRT记录
nslookup 域名 查询A记录
nslookup -type=mx 域名 查询MX记录
nslookup -type=cname 域名查询CNAME记录
nslookup -type=ns 域名 查询域名服务器
nslookup 域名 IP 制定域名服务器查询结果

last 最近登录登出日志
ping IP -c 次数 -s 内容
