0x00介绍

这篇文章是介绍一下在centos操作系统上手动安装ettercap，举一反三，其他软件的安装就不是问题了！(详细演示过程请查看本人博客)

0x04总结：
前边的四个包安装很顺利，关键是执行cmake的时候一定要一步一步来，依赖包很多，要耐得住性子，一个一个来（注意：安装完一个依赖包之后就执行一下rm -rf *将生成的文件删除，以免在执行cmake的时候会报已安装过的包未找到的错误）！

0x05拓展
安装软件不只有yum 一种，是具体的环境而定：
wget：
类似于迅雷，是一种下载工具，通过HTTP、HTTPS、FTP三个最常见的TCP/IP协议下载，并可以使用HTTP代理名字是World Wide Web”与“get”的结合。

yum:
是redhat,centos 系统下的软件安装方式，基于Linux，全称为 Yellow dog Updater,Modified，是一个在Fedora和RedHat以及CentOS中的Shell前端软件包管理器基于RPM包管理，能够从指定的服务器自动下载RPM包并且安装，可以自动处理依赖性关系，并且一次安装所有依赖的软件包。

rpm:
软件管理;redhat的软件格式 rpm ：r=redhat p=package m=management用于安装 卸载 .rpm软件

串联下：
使用wget下载一个 rpm包, 然后用 rpm -ivh xxx.rpm 安装这个软件，嫌麻烦的话，就可以直接用 yum install sqoop来自动下载和安装依赖的rpm软件。

ap-get：
ubuntu下的一个软件安装方式，它是基于debain。
