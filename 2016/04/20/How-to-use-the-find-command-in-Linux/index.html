How to use the find command in Linux

Find 用于查找文件系统中的指定文件
find . -name 1.txt （.当前目录 ..上级目录）查找名字为1.txt（遍历查找）（注意空格）
-name “1.”使用通配符需要加上双引号或者单引号可以再任何位置
部分参数：
-exec 对应的格式是{} \;结束，一定要注意空格以；结束前边没空格
-print 对找到的文件标准输出
-perm 权限查找的时候后边加上权限值
-ok 和-exec用法相同
-name 后跟要查找的文件名
-user username 按照属主来查找
-type b/d/c/p/l/f 查找块设备、目录、字符设备、管道、符号链接、普通文件
-mtime -n +n 按文件更改时间来查找文件，-n指n天以内，+n指n天以前
-atime -n +n 按文件访问时间来查
-ctime -n +n 按文件创建时间来查找文件，-n指n天以内，+n指n天以前
-size 按照大小查找
find /etc/ -name *.conf -print 查找的文件输出出来

find /tmp -name “.”(查找/tmp下面的所有文件)

find /bin/ -perm 4755 | xargs ls -al 把前一个命令的输出转化为后一条命的参数
按照权限查找加上-perm

find /bin/ -perm 4755 -exec ls -al {} \; 命令放在find命令的-exec选项中(注意以；结束)
Exec后面跟执行的命令或脚本

自己感受一下看看有什么不同

find ~ -name “*.txt” -perm 644 -exec ls -al {} \;按照权限名称查找：

find ~ -name “[a-z]*.txt” -perm 644 -exec ls -al {} \;查找以小写字母开头权限为644的文件

find /home -size -/+512k查找小于/大于512k的文件

find / -user xiaoli查找用户小李下的目录：

find / -amin -10 # 查找在系统中最后10分钟访问的文件
find / -atime -2 # 查找在系统中最后48小时访问的文件
find / -empty # 查找在系统中为空的文件或者文件夹
find / -group cat # 查找在系统中属于 groupcat的文件
find / -mmin -5 # 查找在系统中最后5分钟里修改过的文件
find / -mtime -1 #查找在系统中最后24小时里修改过的文件
find / -nouser #查找在系统中属于作废用户的文件
find / -user fred #查找在系统中属于FRED这个用户的文件
参数太多了，一个一个慢慢试吧！
find . -type f -perm +6000就是指定目录下所有具有suid/sgid的程序。
这是在做安全维护的时候，应该常使用的一个指令
