How to use the sed command in Linux

0x00介绍

sed 文本处理工具，本身是一个管道命令主要以行为单位进行处理，可以将数据进行增删改选：对文件的编辑！(详细演示请参见本人新浪博客)

0x01参数介绍​

-n:使用安静模式。

-e:直接在命令行模式运行sed的动作编辑（可以有多个动作）

-f:直接将sed的动作写在一个文件里边：写入一个新的文件（很少用）

-r:sed的动作支持的是延伸正规表示的语法

-i:覆盖（直接诶修改读取的文件类型）

0x02动作说明：

a:增加

c:取代

d:删除

i插入

p:打印

s:取代

0x03示例展示​

nl /etc/passwd | sed ‘2,$d’ （d:delete是删除，2-5行）

nl输出到管道里边 操作的是缓存

nl /etc/passwd | sed ‘2a jiadong’ （a：add是增加，在第二行下边增加内容）

nl /etc/passwd | sed ‘2i aaaaaa’插入是向上插入（i：inster 插入）

nl /etc/passwd | sed ‘2,5c aaaaa’将2-5行替换为aaaaa

nl /etc/passwd | sed -n ‘2,5p’安静输出2-5行

nl /etc/passwd | sed -n ‘/root/p’根据正则表达式列出输出的东西

nl /etc/passwd | sed ‘/root/d’匹配到root的那一行删除掉

nl /etc/passwd | sed -n ‘/root/p’,查找/etc/passwd中包含root串的行并打印出来：

nl /etc/passwd | sed -n ‘/root/{s/root/abcd/g;p}’只在有/root的行里边替换root输出abcd ，·/g全局替换

nl /etc/passwd | sed -n ‘1,7{s/root/abcd/;p}’1-7行里边替换开头/root

使用正则表达式进行查找ip：

ifconfig | grep ‘inet addr’ | sed -n ‘{s/^.addr/ip/g;p}’ |sed -n ‘{s/ .$//g;p}

0x04总结

sed主要进行的是​文本编辑操作，可以通过特定的查找，结合正则表达式就可以实现自己想要的功能，sed所面向的是字节流的，上面介绍的只是自己一个晚上了解的皮毛而已，学不可以已、、、、、
